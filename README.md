## Project Overview
This project analyzes a library database using SQL to understand book availability,
borrower activity, and branch-wise loan details.

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Database Tables
- tbl_book
- tbl_book_authors
- tbl_book_copies
- tbl_book_loans
- tbl_borrower
- tbl_library_branch

## Key Analysis
- Book availability by branch
- Borrowers with no loans
- Branch-wise loan counts
- Frequent borrowers
- Author-wise book analysis

## Skills Demonstrated
- SQL JOINs
- WHERE clause
- GROUP BY & HAVING
- LEFT JOIN
- Date filtering
